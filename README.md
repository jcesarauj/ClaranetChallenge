## General info
Este projeto é uma aplicação web desenvolvida em Angular no frontend e .NET Core no backend, que permite o gerenciamento de cadastro de newsletter.

## Features

- Registration of new users to receive the newsletter.
	
## Technologies

- Node.js
- Angular CLI
- .NET Core SDK
